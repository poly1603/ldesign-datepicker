## 🚀 如何使用

### 快速开始

```bash
pnpm add @ldesign/datepicker-vue
```

```vue
<template>
  <DatePicker v-model="date" type="date" />
</template>

<script setup>
import { ref } from 'vue';
import { DatePicker } from '@ldesign/datepicker-vue';

const date = ref(new Date());
</script>
```
